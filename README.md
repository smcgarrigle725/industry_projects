# industry_projects

Data science and data engineering projects using public industry datasets in healthcare, finance, and insurance. Each project is scoped around a realistic business problem and demonstrates the full workflow from data ingestion through modeling, evaluation, and communication of results.

## Structure

Each project folder follows a consistent layout:

```
project_name/
├── README.md          # problem statement, approach, results, business framing
├── data/              # data sources, download instructions (no raw data committed)
├── notebooks/         # exploratory analysis and modeling (Jupyter or R Markdown)
├── src/               # reusable scripts and modules
└── outputs/           # figures, tables, model summaries
```
